# AI Stack Extras

Helper assets and public usage-count summaries for [Self-Hosted AI Stack](https://github.com/hwdsl2/self-hosted-ai-stack) and related AI Docker images.

Self-Hosted AI Stack and selected AI Docker images use public GitHub release asset download counts for anonymous, aggregate usage counts. These counts help understand usage and prioritize future improvements. The projects do not send a telemetry payload or use a private collector.

Current summaries can be found in the [stats](stats) folder.
